# 历史内容策略

> 状态：基线已确认（2026-07-12）。关联决策：D004、D005、D010、D020。

## 首个历史切片：赤壁前后

- 范围：公元 208 年曹操南下至赤壁战后格局初定。
- 地域：荆州、长江中游及江东相关区域。
- 主要势力：曹操、孙权、刘备三方。
- 核心行动者：曹操、孙权、刘备、周瑜、诸葛亮。
- 选择理由：时间跨度可控、行动者关系清晰、战略变量丰富、用户认知基础较高、远离近现代敏感政治。
- MVP 不模拟：后世完整三国格局、民间演义中的超自然情节、无法验证的私人对话。

## 史料政策

- 一手核心：以开放许可或可合法引用版本的《三国志》相关纪传为核心主张来源。
- 二手补充：经过同行评审的历史研究、权威出版社注释和可靠考古资料。
- 文学材料：《三国演义》等只能作为接受史或叙事对照，不作为史实基线。
- 每个史料主张记录来源、版本、位置、摘录边界、解释、置信度和冲突主张。
- 置信度使用“高 / 中 / 低 / 争议”四级，并说明理由。
- 来源冲突不强行合并；基线选择保留反方证据。

## 内容制作流程

1. 定义切片时间、地域和问题范围。
2. 建立来源目录和许可证清单。
3. 抽取最小史料主张并交叉核验。
4. 由历史内容负责人确定基线版本。
5. 将主张映射为行动者观察、资源、关系和制度约束。
6. 进行历史审阅、规则审阅和敏感性审阅。
7. 发布后保留纠错入口；修订产生新基线版本。

## 展示与伦理

- 史料内容标记“历史基线”，模拟内容标记“世界线推演”，AI 文本标记“AI 解释”。
- 来源在事件详情中一跳可达，摘要不能替代原始出处。
- 暴力、死亡和族群冲突按内容分级处理，避免娱乐化现实创伤。
- 不对历史人物进行现代医学或道德诊断；价值判断说明观察视角。
- 用户公开内容需自动检测，涉及高风险内容时进入人工复核。
