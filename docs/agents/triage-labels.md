# Triage Labels

| Canonical role | GitHub label | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 等待维护者评估 |
| `needs-info` | `needs-info` | 等待补充信息 |
| `ready-for-agent` | `ready-for-agent` | 信息完整，可由 AI 独立处理 |
| `ready-for-human` | `ready-for-human` | 需要人工判断或执行 |
| `wontfix` | `wontfix` | 决定不处理 |

技能提到某个角色时，使用表中对应的 GitHub 标签。当前“只做架构与文档”的限制高于标签状态。
