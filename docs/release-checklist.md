# 1.0 发布清单（DEV-031）

> 关联：`docs/quality.md`、D021。部署与云资源须项目所有者单独授权。

## 自动化闸门

- [ ] `npm run check`（typecheck + test + build）通过  
- [ ] `npm audit --omit=dev` 无未解释的高危/严重漏洞  
- [ ] 单元/集成测试覆盖：幂等、超时失败不写库、重启不丢事件  

## 产品验收（D021）

- [ ] 自由输入改写 → AI 推演（桩或真模型）→ 状态更新  
- [ ] 事件日志可回溯；上帝面板与状态一致  
- [ ] 重复 `commandId` 不重复推进  
- [ ] 推演内容有「非史实」标识  
- [ ] 替换模型/框架不改变世界状态 schema 语义  

## 安全（1.0 最小）

- [ ] 密钥不在仓库/客户端（仅服务端 env）  
- [ ] 改写有长度与格式校验  
- [ ] 推演有速率限制  
- [ ] 日志无明文 API Key（脱敏）  
- [ ] 响应含基础安全头
