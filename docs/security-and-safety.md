# 安全、隐私与内容治理

> 状态：1.0 基线已确认（2026-07-12）。关联决策：D017、D018、D019、D020。本文不创建部署配置。

## 信任模型

- 浏览器、用户输入、模型输出和外部内容均不可信。
- 后端世界状态写入路径（记录员 Agent）是高信任区域。
- Agent 默认无任意网络/Shell/文件权限，仅可读写世界状态数据库与事件日志（D018）。

## Agent 权限（1.0）

- Agent 默认禁止任意网络、Shell、进程和文件系统访问。
- 演员 Agent 只读世界状态；记录员 Agent 写回世界状态数据库与事件日志；二者均不获得系统管理权限。
- 工具参数执行结构与语义校验，绑定当前会话与改写。
- 修改模型、提示词、Agent 配置、权限或世界状态 schema 必须人工审批（D017）。
- 每次模型运行设置 Token、时间与重试预算，可随时取消。
- 外部网络/文件/进程访问若未来启用，应放入独立沙盒并记录 ADR。

## 应用安全（1.0 最小）

- 输入校验、输出编码、速率限制和依赖审计进入发布闸门。
- Prompt injection 内容与系统指令隔离；用户改写文本永不被当作工具指令。
- 密钥只存在于服务端秘密管理，不进入仓库、客户端、日志或模型上下文。
- 身份集成、CSRF/CSP 防护等细节随实现阶段补充；1.0 单会话，不涉及多用户授权。

## 隐私（1.0 最小）

- 只收集故障诊断和滥用防护所需数据。
- 私有会话默认不公开。
- 原始模型会话与审计元数据保留策略留待后续细化（D019）。
- 启用第三方模型前记录其数据保留、训练和地域政策。

## 内容治理（1.0 最小）

- 推演内容不得伪装为史实（D010 原则保留；具体 UI 标识留待后续）。
- 暴力、仇恨、性内容、自残和现实政治煽动按风险分级（细化留待后续，D020）。
- 1.0 单会话不涉及公开分享；公开内容自动检测与人工复核留待后续。
- 提供基础纠错与举报入口（随实现阶段补充）。

## 待定项

- 身份集成与多用户授权方案（1.0 单会话）。
- 原始模型会话与审计元数据保留策略（D019）。
- 内容分级与公开内容复核流程（D020）。
