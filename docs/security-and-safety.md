# 安全、隐私与内容治理

> 状态：基线已确认（2026-07-12）。关联决策：D017、D018、D019、D020。

## 信任模型

- 浏览器、用户输入、历史附件、模型输出和外部内容均不可信。
- 后端领域规则与事件写入路径是高信任区域。
- Pi 提供 Agent 运行能力，但不是权限沙盒；Sandtable 负责所有授权。

## Agent 权限

- 默认禁止任意网络、Shell、进程和文件系统访问。
- Agent 只获得读取合法观察、查询已批准史料和提交行动意图的领域工具，永不获得 Root 改写工具。
- Root 权限只作用于用户拥有的具体世界线；不能修改全局史料、规则、模型、权限或其他用户数据。
- 工具参数执行结构和语义双重校验，并绑定用户、行动者、世界线和模拟时刻。
- 修改史料、规则、模型、提示词、权限和公开内容必须人工审批。
- 每次模型运行设置 Token、时间、工具次数和重试预算，可随时取消。

## 应用安全

- 使用标准 OIDC/OAuth 身份集成，后端使用短期会话和安全 Cookie；移动 Web 不把长期令牌放入 localStorage。
- 所有写操作执行对象级授权，分享链接使用可撤销的随机能力令牌。
- CSP、CSRF 防护、输入校验、输出编码、速率限制和依赖审计进入发布闸门。
- Prompt injection 内容与系统指令隔离；史料文本永不被当作工具指令。
- 密钥只存在于服务端秘密管理，不进入仓库、客户端、日志或模型上下文。

## 隐私

- 只收集账户同步、世界线保存、故障诊断和滥用防护所需数据。
- 私有世界线默认不用于模型训练或公开展示。
- 原始模型会话默认保留 30 天，审计元数据保留 180 天。
- 用户可导出并删除账户和世界线；删除传播到缓存、对象和后续备份过期流程。
- 启用第三方模型前记录其数据保留、训练和地域政策。

## 内容治理

- 产品明确区分历史基线、模拟事件和 AI 解释。
- 暴力、仇恨、性内容、自残和现实政治煽动按风险分级。
- 私人沙盒以提示和限制为主；公开分享执行自动检测，高风险内容人工复核。
- 提供来源纠错、内容举报、处理状态和申诉渠道。
- 严重安全或内容事件暂停分享能力，保留证据并完成复盘。
